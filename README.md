### Setting up venv 
- - -
1. Create a virtual environement
    ```
    py -3.10 -m venv .venv
    source .venv/bin/activate #on mac
    .venv\Scripts\activate #on windows
    ```
2. Install libraries
   ```
   pip install tensorflow matplotlib 
   ```
